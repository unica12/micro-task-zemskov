# Система управления задачами для строительных объектов
## ООО «СистемаКонтроля»

### Обзор проекта
Микросервисная система для управления задачами на строительных объектах.

### Архитектура
1. **API Gateway (api_gateway/)** - Единая точка входа, аутентификация, маршрутизация
2. **Сервис пользователей (service_users/)** - Управление пользователями, аутентификация
3. **Сервис заказов (service_orders/)** - Управление жизненным циклом заказов

### Технологии
- **Среда выполнения**: Node.js 18+
- **Фреймворк**: Express.js
- **Аутентификация**: JWT (JSON Web Tokens)
- **Логирование**: Pino
- **Валидация**: Joi/Zod
- **Контейнеризация**: Docker + Docker Compose
- **Документация API**: OpenAPI 3.0

### Требования из ТЗ
- [ ] JWT аутентификация
- [ ] Ролевая модель (инженер, менеджер, администратор)
